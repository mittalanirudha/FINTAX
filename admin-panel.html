🔥 Firebase Setup Instructions
Pehli baar setup ke liye yeh steps follow karein
Step 1: console.firebase.google.com pe jaao → New Project banao
Step 2: Project mein "Firestore Database" enable karo (Start in test mode)
Step 3: Project Settings → General → "Your apps" → Web app add karo
Step 4: Firebase config copy karo (apiKey, projectId etc.)
Step 5: Dono HTML files mein FIREBASE_CONFIG object mein apna config paste karo
Step 6: Dono files WordPress Media Library ya FTP se upload karo
Step 7: WordPress mein do Pages banao → HTML embed karo (yahan instructions dekhein)

⚠️ Security: Firestore Rules mein sirf authenticated read/write allow karo (admin panel se set hoga) — default test mode 30 din baad expire hoti hai!
