🔥 Data Save Nahi Ho Raha? — Firestore Rules Fix (ZAROOR KAREIN)
Yeh karo aur data save hona shuru ho jaayega
Step 1: Yahan click karo — Firestore Rules directly open honge
Step 2: Saara existing code delete karo
Step 3: Neeche wala code paste karo aur Publish click karo:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
✅ Publish click karte hi data save hona shuru ho jaayega!
Agar pehli baar Firebase setup kar rahe ho: console.firebase.google.com → Project open karo → Firestore Database → Rules
